# Sanguine

A chronological gallery of sanguine drawings by
[Tristan Van Garsse](https://tristanvangarsse.com).

## Live site

Visit [sanguine.vangarsse.com](https://sanguine.vangarsse.com).

## About

Sanguine is a lightweight, responsive archive that presents drawings by year
in a horizontal gallery with smooth wheel scrolling and keyboard navigation.

## Built with

- HTML
- CSS
- Vanilla JavaScript

## Run locally

No build step or dependencies are required. Serve the repository with any
static web server, for example:

```sh
python3 -m http.server 8000
```

Then open [localhost:8000](http://localhost:8000).
